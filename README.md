# rabe

A minimalistic implementation of **R**edls **a**nalytial **b**ootstrap **e**stimation for stellarators [1-3].

## Build

use

```bash
make
```

to build the executable and

```bash
make test
```
to execute tests.

## Third Party

See licenses under `third_party`

- `pyplot-fortran` for visualization
