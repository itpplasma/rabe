# rabe

A minimalistic implementation of **R**edls **a**nalytial **b**ootstrap **e**stimation [1].

## Build

use

```bash
make
```

to build the executable

## References
[1] A. Redl et al., A new set of analytical formulae for the computation of the bootstrap current and the neoclassical conductivity in tokamaks, Phys. Plasmas 28, [doi:10.1063/5.0012664](https://doi.org/10.1063/5.0012664)

[2] O. Sauter et al., Neoclassical conductivity and bootstrap current formulas for general axisymmetric equilibria and arbitrary collisionality regime, Phys. Plasma 6, [doi:10.1063/1.873240](https://doi.org/10.1063/1.873240)
